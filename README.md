# psgC6_neck_visualizer
Pedal Steel Guitar Fretboard Visualizer for C6 tuning

A compact C6 tuning fretboard visualizer on Processing for pedal steel guitar.
The system renders a clear, real-time layout of notes across the my 12-string C6 neck (4 pedals, 4 knee levers), helping players better understand positions, intervals, and chord structures.

Accurate C6 tuning mapping

Ideal for practice, studying theory, or exploring new pedal steel positions directly on the instrument.
